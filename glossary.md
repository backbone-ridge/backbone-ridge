# Glossary

Here below are listed various terms and abbreviations commonly found in the manuscript.

| Term  | Description |
| ----- | ----------- |
| Chs   | Chains (66 ft) |
| cov'd | covered |
| L | links |
| lak | lake |
| lin | linden |
| Lks   | Links (100 links in 1 chain) |
| mark'd | marked |
| N'd   | numbered |
| NE    | northeast |
| NEly  | northeasterly |
| NW    | northwest |
| NWly  | northwesterly |
| p | page |
| pg | page |
| s'd | said or side |
| SE    | southeast |
| SEly  | southeasterly |
| SW    | southwest |
| SWly  | southwesterly |
| Ts'P  | township |
| &c. | etc, et cetera |


## Early writing style

* small `s` looks like an `f`
