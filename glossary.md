# Glossary

Here below are listed various terms and abbreviations commonly found in the manuscript.

| Term  | Description |
| ----- | ----------- |
| Chs   | Chains (66 ft) |
| Cov'd | covered |
| Lks   | Links (100 links in 1 chain) |
| N'd   | numbered |
| NE    | northeast |
| NEly  | northeasterly |
| NW    | northwest |
| NWly  | northwesterly |
| SE    | southeast |
| SEly  | southeasterly |
| SW    | southwest |
| SWly  | southwesterly |
| Ts'P  | township |
